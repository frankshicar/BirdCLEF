## 相較前版（0.858）的改動

### 1. Secondary label soft label（dataset.py）

primary label = 1.0，secondary labels 從 1.0 改為 0.3。

Secondary labels 本來就不確定（錄音中可能有其他鳥但不是主角），用 0.3 比 1.0 更符合實際，減少 label noise 對訓練的干擾。

### 2. Global Mean + Max pooling（model.py）

pooling 從 `avg`（global average pool）改為 `mean_max`（global mean + global max concat）。

EfficientNet-B0 的 feature map 經過 mean 和 max 兩種方式壓縮後 concat，feature dim 從 1280 → 2560。Mean pool 捕捉整體分布，Max pool 捕捉最顯著的局部特徵，兩者互補。

### 3. Gradient clipping（train.py）

加入 `torch.nn.utils.clip_grad_norm_(model.parameters(), 5.0)`。

防止梯度爆炸，讓訓練更穩定，特別是在 mixed precision 下。

### 4. Early stopping 指標改為 mAP（train.py）

從 macro ROC-AUC 改為 macro mAP 作為 checkpoint 儲存和 early stopping 的依據。

mAP 對稀少物種更敏感，更能反映模型在長尾分布上的真實表現。

### 5. Mask denoiser（model.py）

`ResidualDenoiser` 從加法模型改為乘法 mask 模型：

```
舊版: output = input - predicted_noise   # 可能過度減去
新版: mask = sigmoid(conv4(features))    # 值域 [0, 1]
      output = input * mask              # 只能抑制，不會過度減去
```

Mask 的值域限制在 [0,1]，denoiser 只能選擇保留或抑制頻率，不會「憑空減去」任何東西，避免把鳥叫聲也一起濾掉。

### 6. n_mels 128 → 160，n_fft 1024 → 2048

更細緻的頻率解析度，160 mel bins 能區分更相似的鳥種叫聲。n_fft 從 1024 升到 2048（freqs 從 513 → 1025）以避免 mel filterbank 零值警告。

## 貢獻分析

本版同時改了多個項目，無法單獨歸因，但推測貢獻排序：

1. **Mask denoiser** — 解決 denoiser 學歪的根本問題，預估貢獻最大
2. **Secondary soft label 0.3** — 減少 label noise，對稀少物種特別有效
3. **Mean + Max pooling** — 更豐富的特徵表示
4. **n_mels 160** — 更細緻的頻率解析度
5. **Gradient clipping + mAP early stopping** — 訓練穩定性改善
